## 一、JavaScript 文件（.js）

.js 文件是 JavaScript 文件的扩展名，主要用于存储和执行 JavaScript 代码。

### 作用

- **实现交互效果**：响应用户操作（点击、移动鼠标等），改变页面内容或样式
- **动态修改页面内容**：创建、修改或删除 HTML 元素
- **表单验证**：提交前验证数据合法性和完整性
- **数据操作**：处理 JSON、XML 等格式数据
- **与服务器通信**：通过 AJAX 实现异步通信
- **构建复杂应用**：结合 HTML、CSS 构建 Web 应用（如 SPA）

## 二、时间复杂度

时间复杂度用来衡量算法运行时间和输入规模的关系，用大 O 符号表示。

### 表示方法

| 符号 | 名称 | 说明 |
|------|------|------|
| O(n) | 大O符号 | 最坏情况下，算法的时间复杂度 |
| o(n) | 小O符号 | 严格小于另一个算法 |
| Ω(n) | 大Ω符号 | 最好情况下，算法的时间复杂度 |
| ω(n) | 小ω符号 | 严格大于另一个算法 |
| Θ(n) | Θ符号 | 确切时间复杂度 |

### 常见复杂度

| 复杂度 | 名称 |
|--------|------|
| O(1) | 常数时间 |
| O(log n) | 对数时间 |
| O(n) | 线性时间 |
| O(n²) | 平方时间 |

> 大 O 符号最常用，描述最坏情况下的性能。

## 四、深度学习 - RDB 网络结构

RDB（Residual Dense Block，残差密集块）是一种深度学习网络结构。

### 特点

- 结合**残差连接**和**密集连接**
- 缓解深层网络梯度消失问题
- 增强特征复用和信息流动

### 结构

- 每个残差密集层由多个卷积层组成
- 卷积层输出通过残差连接与块输入结合
- 所有层输出通过密集连接汇总

### 应用

图像分类、目标检测、人脸识别等计算机视觉任务。

## 五、人脸识别 - LFW 数据集

LFW（Labeled Faces in the Wild）是用于人脸识别研究的数据集。

### 概述

- 包含超过 13,000 张人脸图像
- 涵盖各种姿态、光照条件和表情变化
- 每个人脸标记一个人名

### 用途

- 评估人脸识别算法性能
- 基准测试，比较不同算法准确率和鲁棒性

### 注意事项

- 图像质量和标注可能存在挑战
- 需要注意数据预处理和标注准确性

## 六、Selenium 显式等待

Selenium 中使用显式等待查找子元素并等待出现。

### 示例代码

```python
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

driver = webdriver.Chrome()
driver.get("http://example.com")

wait = WebDriverWait(driver, 10)

try:
    # 等待父元素可见
    parent_element = wait.until(EC.visibility_of_element_located((By.ID, "parent-id")))

    # 在父元素内查找子元素
    child_elements = parent_element.find_elements(By.CLASS_NAME, "child-class")

    if child_elements:
        print("找到子元素")
        child_elements[0].click()
    else:
        print("未找到子元素")
except Exception as e:
    print("错误:", e)
finally:
    driver.quit()
```

### 关键点

- `WebDriverWait` 配合 `until` 等待条件
- `visibility_of_element_located` 等待元素可见
- `find_elements` 返回元素列表，需检查是否为空

## 八、Web 字体格式（.woff2）

.woff2 是 Google 开发的 Web 字体格式，是 .woff 的改进版本。

### 优点

- **更小文件**：压缩率更高，加载更快
- **跨浏览器支持**：Chrome、Firefox、Safari、Edge 均支持
- **开源免费**：无需许可费用
- **字体保护**：更好防止非法使用

### 使用方法

```css
@font-face {
    font-family: 'MyFont';
    src: url('myfont.woff2') format('woff2'),
         url('myfont.woff') format('woff');
    font-weight: normal;
    font-style: normal;
}

body {
    font-family: 'MyFont', sans-serif;
}
```
