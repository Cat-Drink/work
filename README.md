# 基于 Zensical 的博客站
这是一个基于 Zensical 构建的博客站，用于记录自己的学习过程。

> 本项目在开发过程中，参考了 [Wcowin.github.io](https://github.com/Wcowin/Wcowin.github.io) 以及[Zensical中文教程](https://wcowin.work/Zensical-Chinese-Tutorial/)的项目结构和代码逻辑。特此感谢原作者 [Wcowin] 的杰出工作。
